How the AI Works
📡
Phase 1: N8N Learning
Sends requests to n8n. Records all donor responses and outcomes to build training data
📊
Phase 2: Score Building
Assigns availability, reliability, and response scores to each donor using outcomes data
🚀
Phase 3: Autonomous
After 50 matches, AI bypasses n8n — directly matches from its own verified donor database

N8N Payload Format
{
  "request_id": 42,
  "blood_group": "O+",
  "units_needed": 2,
  "urgency": "urgent",
  "hospital_name": "Lagos General Hospital",
  "hospital_city": "Lagos",
  "hospital_phone": "+234801...",
  "callback_url": "https://yourapp.com/api/webhook/n8n/donors-found/"
}
N8N responds to the callback_url with a list of matched donors.
